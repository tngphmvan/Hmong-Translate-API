# Hmong-Translate-API

A cloud-based translation service providing bidirectional translation capabilities between Hmong and other languages (English, Vietnamese).

## Documentation

- **[System Architecture](ARCHITECTURE.md)**: Comprehensive architecture overview including components, data flow, deployment, and diagram descriptions for visualization.

## Overview

This API enables translation between Hmong and other major languages, supporting both individual and batch translations with a focus on accuracy and cultural context.

## Features

- Bidirectional translation (Hmong ↔ English, Hmong ↔ Vietnamese)
- RESTful API with comprehensive documentation
- Custom dictionary support
- Translation caching for improved performance
- User authentication and API key management
- Usage analytics and reporting
